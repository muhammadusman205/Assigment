HTML Questions and Answers

Q1: What is HTML?
Answer: HTML is a markup language used to create web pages.

Q2: What is the purpose of DOCTYPE?
Answer: It tells the browser about the HTML version.

Q3: What are semantic tags?
Answer: Tags that describe the meaning of content.
